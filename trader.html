Create a Binance account

https://www.binance.com


If you need guidance, follow the Binance Beginner's Guide to help get started with your account:

Binance Beginners Guide



Open a Futures account

You will need KYC verification to open your futures account on Binance:

https://www.binance.com/futures



Create an API for your Binance account

Go to your profile highlighted by the red box.




Select the Account tab and then API Management highlighted by the orange boxes.

Click on the Create API yellow button highlighted by the red box.




On the Choose API Key type menu select the System generated option.




Set a name of your choosing to the Binance API.

Click Next and complete your 2-step verification to finalize API creation.




Save your API Key and your Secret Key, you'll only be able to see your Secret Key the first time you create the API.

Click on Edit restrictions highlighted by the red box.




Select the option “Restrict access to trusted IPs only (Recommended)” highlighted by the red box.

Type the following IP address: 152.42.138.245 in the yellow box

Check the box that says Enable Futures highlighted by the orange box.

Click the Save button on the top right and that's it, the API is now set to allow Futures trading.




Configuring Binance Futures trading settings for BTC

Under the Futures tab highlighted by the red box select USDⓈ-M Futures highlighted by the orange box.




Select the BTCUSDT pair highlighted by the orange box.

Change your settings to match the settings highlighted by the yellow box. It should be Cross Margin Mode and 100x Leverage.

Click on the settings button on the top right of the page highlighted by the red box.




Select the tab Position Mode highlighted by the red box.




Choose the option One-Way Mode if it's not already selected.




Connecting to the Binance API

Go to the Crypto Trading Flow website:

https://www.cryptotradingflow.com

Log In to your Crypto Trading Flow account or if you don't have an account click on Sign Up highlighted by the red box.




Select the Exchanges sidebar tab highlighted by the orange box.

Click the + Add Exchange button highlighted by the red box.




Fill the exchange API form with your Binance API keys and click the Save API button highlighted by the red box.




You're Binance API will be displayed in the table if added successfully.




All that's left is to select a trading bot to copy-trade.

Select the Copy-Trading sidebar tab highlighted by the orange box.

Click on the Copy-Trade button highlighted by the red box.




If everything is set up correctly you should now see the corresponding trading bot in the Active Bots section.




In order to copy-trade other coins, just follow these same steps and adjust the leverage to match the corresponding CTF bot leverage.


Don't forget you'll have to transfer funds from your Spot wallet to your Futures wallet.

If you need help setting up your Futures account you can check the following guide:

The Ultimate Guide to Trading on Binance Futures

🔐 Security & Privacy

- Is it safe to share my API keys with CTF?

Yes. Your API keys are encrypted and securely stored. We only request Futures trading permissions—withdrawals are never required.

- Is it safe to share my API keys with CTF?

Only enable Futures Trading. Leave Withdrawals disabled for security.

⚙️ Bot Features

- How often do the bots trade?

It depends on the strategy and market conditions. Some bots trade several times a day, others less frequently.

- Can I stop copy-trading at any time?

Yes. Just cancel the bot you want to stop under the Active Bots section in CTF Copy-Trading.

⚠️ If you have active trades, you must close them manually on the exchange.

- Do the bots use stop-loss or risk management?

Yes. The bots apply dynamic risk management to limit losses and control drawdowns.

💸 Fees & Costs

- Do I have to pay to use the CTF bots?

No. There's currently no subscription fee—you only pay standard trading fees charged by the exchange.

- Are there performance fees on profits?

No. CTF does not take a share of your profits.

📈 Performance & Results

- How can I view my performance?

Visit CTF Analytics to view your trade history, profit charts, and performance data.

- Are the bot strategies backtested?

Yes. All strategies are backtested and monitored in real time.

📉 Past results do not guarantee future performance.

🚀 Getting Started

- What do I need in order to start copy-trading?

A Futures account with a supported exchange. Some exchanges may require KYC verification.

- How do I start copy-trading a CTF bot?

Follow the CTF User Manual to get started.

- What exchanges are supported?

Currently, only Binance is supported. More exchanges will be added in the future.

- What's the minimum amount I need to have?

Minimum trade requirements:

BTCUSDT: 0.002 BTC

ETHUSDT: 0.011 ETH

⚠️ Insufficient balances may prevent trades and limit performance due to fewer entries.
