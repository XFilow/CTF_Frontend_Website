Open a Futures account

You will need KYC verification to open your futures account on Binance:

https://www.binance.com/futures


Create an API for your Binance account

Go to your account page and select the API Management tab.

Click on the yellow button that says “Create API”.



On the Choose API Key type menu select the System generated option.



Set a label to your Binance API.

Complete your 2-step verification to finalize API creation.



The API screen will have all boxes unchecked except Enable Reading.

Save your API Key and your Secret Key, you'll need them later.

Click on Edit restrictions.



Select the option “Restrict access to trusted IPs only (Recommended)”.

Type the following IP address: 1234567

Check the box that says “Enable Futures”.

Click Save and that's it, the API is set.


Connecting to the Binance API

Login to your Crypto Trading Flow account:

https://www.cryptotradingflow.com/login

Select the Exchanges sidebar tab.

Click the + Add Exchange button.



Fill the exchange API form with your Binance API keys and click the Save API button.


You're Binance API will be displayed in the table if added successfully.

To verify the exchange connection, select the Dashboard sidebar tab and check if your Portfolio Value matches your Binance account.



All that's left is to select a trading bot to copy-trade.

Select the Copy-Trading sidebar tab.

Choose one of the CTF trading bots and start copy-trading.


Configuring Binance Futures account settings for BTC/USDT

Under the Trade tab select USDⓈ-M Futures.



In the Futures screen select the BTC/USDT pair highlighted by the orange box.

Change your settings to match the settings highlighted by the yellow box. It should be Cross Margin Mode and 100x leverage.

Click on the settings button highlighted by the red box.



Select the tab Position Mode.



Choose the option One-Way Mode if it's not already selected.

Everything is set up it's time to let the bot do the trading for you.

Don't forget you'll have to transfer funds from your Spot wallet to your Futures wallet.

If you need help setting up your Futures account you can check th following guide:

The Ultimate Guide to Trading on Binance Futures

What is Crypto Trading Flow?

Crypto Trading Flow is a community focused on leveraging automated bots to trade cryptocurrencies. We offer a variety of trading bots, ranging from free to paid options, with different cryptocurrency pairs.


How can I contact support?

You can ask for support on the discord channel or you can send an email to support@cryptotradingflow.com


What do I need in order to start running a CTF bot?

Using our spot trading bots doesn't have any requirement. To use our futures trading bots however, you'll need a KYC-verified Binance Futures account.


How do I start using a CTF bot?

Click the Getting Started button in the sidebar and follow the setup guide.


What exchanges are supported?

Currently, we only work with the Binance exchange. However, more exchanges will be added in the future.


What's the minimum amount I need to have in my account?

You'll only need to have a minimum amount, according to the pair, if you're using the futures trading bot.


Is there a risk I could lose all my money?

Yes, although unlikely, it can always happen as the markets are constantly changing, and the bots past performance doesn't guarantee future results.


How do I get the Flow Whale role in Discord?

If you've made a donation to one of the CTF wallet addresses you can paste the transaction ID on the whale-role discord channel and wait for verification.
